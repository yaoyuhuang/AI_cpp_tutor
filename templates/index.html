可视化学习：排序算法

通过柱状图观察排序过程，黄色表示正在比较，红色表示交换，绿色表示已经确定位置。适合配合 C++ 排序代码理解循环、下标和交换逻辑。

选择排序方法后，可以单步观察每一次比较和交换。

排序方法理解

冒泡排序

相邻元素两两比较，大的逐步“冒泡”到右侧。重点看双重循环和相邻交换。

for (int i = 0; i < n - 1; i++)
    for (int j = 0; j < n - 1 - i; j++)
        if (a[j] > a[j + 1]) swap(a[j], a[j + 1]);

选择排序

每轮从未排序区间中选择最小值，放到当前轮起点。

for (int i = 0; i < n; i++) {
    int p = i;
    for (int j = i + 1; j < n; j++)
        if (a[j] < a[p]) p = j;
    swap(a[i], a[p]);
}

插入排序

左侧维护有序区，把当前元素插入到合适位置。适合理解“移动元素”。

for (int i = 1; i < n; i++) {
    int x = a[i], j = i - 1;
    while (j >= 0 && a[j] > x) a[j + 1] = a[j--];
    a[j + 1] = x;
}

快速排序

选一个基准值，把小的放左边，大的放右边，再递归处理左右区间。

void quick_sort(int l, int r) {
    if (l >= r) return;
    int i = l, j = r, x = a[(l + r) / 2];
    while (i <= j) {
        while (a[i] < x) i++;
        while (a[j] > x) j--;
        if (i <= j) swap(a[i++], a[j--]);
    }
    quick_sort(l, j);
    quick_sort(i, r);
}

堆排序

先建大根堆，把最大值不断交换到末尾，再缩小堆范围。

void down(int i, int n) {
    while (true) {
        int t = i, l = i * 2 + 1, r = i * 2 + 2;
        if (l < n && a[l] > a[t]) t = l;
        if (r < n && a[r] > a[t]) t = r;
        if (t == i) break;
        swap(a[i], a[t]);
        i = t;
    }
}
for (int i = n / 2 - 1; i >= 0; i--) down(i, n);
for (int end = n - 1; end > 0; end--) {
    swap(a[0], a[end]);
    down(0, end);
}

希尔排序

按 gap 分组做插入排序，gap 逐渐缩小到 1，是插入排序的改进版。

for (int gap = n / 2; gap > 0; gap /= 2) {
    for (int i = gap; i < n; i++) {
        int x = a[i], j = i;
        while (j >= gap && a[j - gap] > x) {
            a[j] = a[j - gap];
            j -= gap;
        }
        a[j] = x;
    }
}

归并排序

先递归拆分，再把两个有序区间合并。稳定排序，时间复杂度 O(n log n)。

void merge_sort(int l, int r) {
    if (l >= r) return;
    int mid = (l + r) / 2;
    merge_sort(l, mid);
    merge_sort(mid + 1, r);
    int i = l, j = mid + 1, k = 0;
    while (i <= mid && j <= r)
        tmp[k++] = a[i] <= a[j] ? a[i++] : a[j++];
    while (i <= mid) tmp[k++] = a[i++];
    while (j <= r) tmp[k++] = a[j++];
    for (int t = 0; t < k; t++) a[l + t] = tmp[t];
}

OI Wiki 中文算法竞赛知识库，适合系统学习数据结构、图论、数论、DP 等。 USACO Guide 结构化算法学习路线，按 Bronze / Silver / Gold / Platinum 分层训练。 cppreference C++ 标准库与语言特性权威参考，适合查头文件、函数、复杂度和标准版本。 cplusplus.com Reference 较易读的 C++ 标准库参考，适合初学者快速查用法。

面试刷题与题单

LeetCode 中文站 面试算法题平台，适合数组、链表、二叉树、动态规划、数据库等专题训练。 AcWing 中文算法学习与题库平台，适合按课程和算法模板系统训练。 Virtual Judge 聚合多 OJ 的训练平台，适合组题、补题和团队训练。

建议学习路线

入门阶段：洛谷入门题单 + AtCoder ABC A-D。

提高阶段：Codeforces Div.3 / Div.2 A-C + OI Wiki 专题。

系统训练：USACO Guide 按等级推进，配合本助手的错题本复盘。

面试方向：LeetCode 中文站按数组、哈希、双指针、栈队列、树、DP 专题刷。
